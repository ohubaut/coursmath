Note technique


N'oubliez pas d'activer, si ce n'est déjà fait, le javascript !

Une lecture confortable nécessite un affichage d'au moins 800 × 600.
Toutes les pages sont écrites en Html5 et utilisent pour les formules mathématiques (rendues soit sous TeX soit en MathML. Tout est parfaitement lisible sous la plupart des navigateurs (IE $ \ge $ 6, Firefox $ \ge $ 3.0, Chrome $ \ge $ 0.3, Safari $ \ge $ 2.0).

Suite à l'abandon de Java par les constructeurs de tablettes, les animations Java qui illustraient en particulier la section "En 3 dimensions" ont été remplacées, par de équivalents qui utilisent WebGL (pour IE $ \ge 11 $, Firefox $ \ge 25 $, Chrome $ \ge 31 $ Opera $ \ge 18 $ ou Safari $ \ge 7 $).
Elles fonctionnent également, mais de manière moins fluide, sous Canvas (pour IE $ \ge 8 $, iOS Safari $ \ge 3.2 $ ou Android $ \ge 2.1 $).

Pour les utilisateurs de McIntosh, nous ne garantissons pas la lisibilité sous n'importe quel OS.
